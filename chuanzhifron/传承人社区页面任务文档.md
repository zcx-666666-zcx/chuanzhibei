  ·1# 传承人社区页面任务文档

## 1. 概述

本文档旨在为团队成员分配传承人社区页面开发相关的具体任务，确保团队协作高效有序。文档将详细说明各项任务的具体内容、操作方法以及任务对最终产品的影响。

## 2. 图片素材收集任务

### 2.1 传承人头像收集
**任务描述：** 收集至少20位代表性非遗传承人的高清头像照片

**具体内容：**
1. 传统技艺类传承人（8位）：景泰蓝、苏绣、蜀锦、景德镇瓷器、紫砂陶、剪纸、木版年画、漆器
2. 传统音乐类传承人（3位）：古琴、琵琶、二胡
3. 传统戏剧类传承人（4位）：京剧、昆曲、越剧、黄梅戏
4. 曲艺类传承人（3位）：相声、评书、快板
5. 民俗类传承人（2位）：春节习俗、端午节习俗

**操作指南：**
1. 通过搜索引擎、非遗保护官网、文化机构网站等渠道寻找合适的高清照片
2. 确保照片分辨率不低于400px宽度
3. 优先选择正面免冠照片，面部清晰可见
4. 注意版权问题，优先选用免费可商用或已获得授权的照片
5. 将照片统一命名为：master_1.jpg, master_2.jpg ... master_20.jpg

**为什么要做这项工作：**
传承人是非遗文化的承载者，其形象展示有助于增强用户对非遗传承的认知和尊重，高质量的头像能提升页面的专业性和可信度。

**对应界面效果：**
在传承人展示区域横向滑动展示，每位传承人配有头像、姓名和荣誉称号。

**具体内容：**

4. 昆曲唱腔教学视频缩略图
5. 古琴演奏教学视频缩略图
6. 剪纸技艺展示视频缩略图
7. 紫砂陶制作教学视频缩略图
8. 景泰蓝制作工艺视频缩略图
9. 蜀锦织造技艺视频缩略图
10. 景德镇瓷器制作视频缩略图
11. 二胡演奏技法视频缩略图
12. 相声表演技巧视频缩略图
13. 评书艺术讲解视频缩略图
14. 快板表演教学视频缩略图
15. 春节习俗介绍视频缩略图

**操作指南：**
1. 从视频平台截图或寻找相关视频的封面图
2. 确保图片分辨率不低于800px宽度

**为什么要做这项工作：**

**对应界面效果：**

### 2.3 用户头像收集
**任务描述：** 收集10张社区用户头像图片

**具体内容：**
1. 收集不同年龄、性别、职业的用户头像各若干张

**操作指南：**
1. 使用免费的头像素材网站获取
2. 确保图片分辨率不低于200px宽度
3. 选择表情自然、清晰度高的头像

用户头像是社区动态的重要组成部分，真实多样的头像能够增强社区的活跃氛围和用户归属感。

**对应界面效果：**
在社区动态区域每条动态左侧显示用户头像。

**具体内容：**

2. 确保图片分辨率不低于800px宽度
3. 图片内容应体现活动特色和现场氛围
4. 注意版权问题，优先选用免费可商用或已获得授权的图片
5. 将图片统一命名为：activity_1.jpg, activity_2.jpg ... activity_5.jpg

**对应界面效果：**

## 3. 文案内容撰写任务

### 3.1 传承人信息撰写
**任务描述：** 为每位传承人撰写基本信息

**具体内容：**
1. 为20位传承人分别撰写准确的姓名
2. 确定每位传承人擅长的非遗技艺
3. 收集每位传承人的主要荣誉称号
4. 为每位传承人撰写一段100字以内的简介

**操作指南：**
1. 姓名以官方公布为准
2. 擅长技艺要准确具体
3. 荣誉称号要选取最具代表性的1-2个
4. 简介要突出传承人的技艺特色和贡献

**为什么要做这项工作：**
准确的传承人信息是对非遗文化工作者的尊重，也是用户了解传承人背景的基础。

**对应界面效果：**
在传承人卡片上显示姓名、擅长技艺、荣誉称号和简介。

### 3.2 视频信息撰写
**任务描述：** 为每个视频撰写相关信息

**具体内容：**
1. 为15个视频分别撰写准确的标题
2. 为每个视频撰写一段50字以内的描述
3. 确定每个视频的播放时长
4. 确定每个视频的发布日期
5. 估算每个视频的播放量

**操作指南：**
1. 标题要简洁明了，能够准确反映视频内容
2. 描述要突出视频的特色和看点
3. 时长和日期要符合实际情况
4. 播放量可根据视频受欢迎程度合理估算

**为什么要做这项工作：**
完整的视频信息有助于用户快速了解视频内容，提升观看决策效率。

**对应界面效果：**
在视频卡片上显示标题、描述、时长、日期和播放量。

### 3.3 活动信息撰写
**任务描述：** 为每个活动撰写相关信息

**具体内容：**
1. 为5个活动分别撰写准确的标题
2. 确定每个活动的具体时间
3. 确定每个活动的举办地点
4. 为每个活动撰写一段100字以内的介绍
5. 确定每个活动的参与方式（报名参加/预约观看）

**操作指南：**
1. 标题要突出活动主题和特色
2. 时间和地点要具体明确
3. 介绍要涵盖活动内容和参与价值
4. 参与方式要清晰明了

**为什么要做这项工作：**
详细的活动信息有助于用户了解活动详情，提升参与积极性。

**对应界面效果：**
在活动卡片上显示标题、时间、地点、介绍和参与按钮。

### 3.4 社区动态文案撰写
**任务描述：** 撰写社区动态内容

**具体内容：**
1. 撰写10条社区动态内容
2. 为每条动态确定发布用户名
3. 为每条动态确定发布时间
4. 为每条动态配上相关图片（1-3张）
5. 为每条动态确定点赞数和评论数

**操作指南：**
1. 动态内容要与非遗文化相关
2. 用户名要多样化，体现不同用户群体
3. 发布时间要合理分布，体现活跃度
4. 图片要与动态内容相关
5. 点赞数和评论数要合理估算

**为什么要做这项工作：**
真实的社区动态能够营造活跃的社区氛围，增强用户参与感和归属感。

**对应界面效果：**
在社区动态区域展示，每条动态包含用户信息、内容、图片和互动数据。

## 4. 数据整理与格式化任务

### 4.1 传承人数据整理
**任务描述：** 将收集到的传承人信息整理成结构化数据

**具体内容：**
1. 建立传承人信息表格，包含字段：ID、姓名、技艺、荣誉称号、简介、头像
2. 确保每个字段信息完整准确
3. 按技艺类型进行分类标记

**操作指南：**
1. 使用Excel或Google Sheets建立信息表格
2. 每位传承人一行，字段列为不同属性

## 5. 图片与数据关联任务

### 5.1 建立图片与数据对照表
**任务描述：** 建立所有图片与对应数据的关联关系

**具体内容：**
1. 建立传承人头像与传承人数据的对照表
2. 建立视频缩略图与视频数据的对照表
3. 建立用户头像与动态数据的对照表
4. 建立活动图片与活动数据的对照表

**操作指南：**
1. 使用Excel或Google Sheets建立对照表格
2. 每张图片一行，包含图片名称、对应数据ID、用途说明等字段
3. 核对关联关系的准确性
4. 整理图片版权相关信息

**为什么要做这项工作：**
确保每张图片都能正确显示在对应的数据项中，避免错位或缺失。

**对应界面效果：**
用户在浏览页面时能看到正确的配图。
