# 个人中心页面任务文档

## 1. 概述

本文档旨在为团队成员分配个人中心页面开发相关的具体任务，确保团队协作高效有序。文档将详细说明各项任务的具体内容、操作方法以及任务对最终产品的影响。

## 2. 图片素材收集任务

### 2.1 用户头像收集
**任务描述：** 收集10张不同类型用户的头像图片

**具体内容：**
1. 收集不同年龄、性别、职业的用户头像各若干张

**操作指南：**
1. 使用免费的头像素材网站获取
2. 确保图片分辨率不低于200px宽度
3. 选择表情自然、清晰度高的头像
4. 注意版权问题，确保可用于商业用途
5. 将图片统一命名为：user_avatar_1.jpg, user_avatar_2.jpg ... user_avatar_10.jpg

**为什么要做这项工作：**
用户头像是个人中心的重要组成部分，真实多样的头像能够增强用户的归属感和认同感。

**对应界面效果：**
在个人中心顶部用户信息区域显示用户头像。

### 2.2 收藏项目图片收集
**任务描述：** 收集至少15个非遗项目的高清图片用于收藏展示

**具体内容：**
1. 传统技艺类项目（5个）：景泰蓝、苏绣、蜀锦、景德镇瓷器、紫砂陶
2. 传统美术类项目（3个）：国画、书法、篆刻
3. 传统音乐类项目（2个）：古琴、琵琶
4. 传统戏剧类项目（3个）：京剧、昆曲、越剧
5. 民俗类项目（2个）：春节、端午节

**操作指南：**
1. 通过搜索引擎、非遗保护官网、文化机构网站等渠道寻找合适的高清图片
2. 确保图片分辨率不低于600px宽度
3. 图片内容应突出项目特色
4. 注意版权问题，优先选用免费可商用或已获得授权的图片
5. 将图片统一命名为：collection_1.jpg, collection_2.jpg ... collection_15.jpg

**为什么要做这项工作：**
收藏项目图片是用户收藏内容的直观展示，高质量的图片能够提升用户对自己收藏内容的满意度。

**对应界面效果：**
在收藏内容区域展示，每张图片配有项目名称、级别和简短描述。

### 2.3 创作作品图片收集
**任务描述：** 收集至少10张用户创作的作品图片

**具体内容：**
1. 剪纸作品（2张）
2. 刺绣作品（2张）
3. 书法作品（2张）
4. 绘画作品（2张）
5. 摄影作品（2张）

**操作指南：**
1. 通过艺术作品网站或自行创作获取
2. 确保图片分辨率不低于600px宽度
3. 图片内容应体现非遗相关元素
4. 注意版权问题，确保可用于展示
5. 将图片统一命名为：creation_1.jpg, creation_2.jpg ... creation_10.jpg

**为什么要做这项工作：**
创作作品图片是用户展示自己学习成果的载体，优秀的图片能够增强用户的成就感和分享欲。

**对应界面效果：**
在创作展示区域以网格形式展示，每张图片配有作品标题。

### 2.4 传承人头像收集
**任务描述：** 收集至少8位传承人的头像图片

**具体内容：**
1. 景泰蓝传承人头像
2. 苏绣传承人头像
3. 蜀锦传承人头像
4. 京剧传承人头像
5. 昆曲传承人头像
6. 古琴传承人头像
7. 剪纸传承人头像
8. 陶瓷传承人头像

**操作指南：**
1. 通过搜索引擎、非遗保护官网等渠道寻找合适的高清照片
2. 确保照片分辨率不低于200px宽度
3. 优先选择正面免冠照片，面部清晰可见
4. 注意版权问题，优先选用免费可商用或已获得授权的照片
5. 将照片统一命名为：master_1.jpg, master_2.jpg ... master_8.jpg

**为什么要做这项工作：**
传承人头像是预约记录的重要组成部分，能够让用户快速识别预约的传承人。

**对应界面效果：**
在预约记录区域每条记录左侧显示传承人头像。

## 3. 文案内容撰写任务

### 3.1 用户信息文案撰写
**任务描述：** 撰写用户信息相关文案

**具体内容：**
1. 确定用户身份标签（如"非遗文化爱好者"、"传统工艺学习者"等）
2. 撰写不同用户类型的标签文案（至少5种）

**操作指南：**
1. 标签要体现用户与非遗文化的关系
2. 语言要亲切友好，避免过于正式
3. 考虑不同用户群体的特点

**为什么要做这项工作：**
用户身份标签能够增强用户的归属感和认同感，体现平台的个性化特色。

**对应界面效果：**
在用户信息区域头像下方显示用户身份标签。

### 3.2 收藏项目信息撰写
**任务描述：** 为每个收藏项目撰写相关信息

**具体内容：**
1. 为15个收藏项目分别撰写准确的项目名称
2. 确定每个项目的保护级别（如"国家级非物质文化遗产"）
3. 为每个项目撰写一段100字以内的简短描述

**操作指南：**
1. 项目名称要准确权威，可参考官方表述
2. 保护级别要核实准确
3. 简介要突出该项目的核心特色和文化价值

**为什么要做这项工作：**
准确的项目信息是用户了解收藏内容的基础，有助于用户管理和回顾自己的收藏。

**对应界面效果：**
在收藏项目卡片上显示项目名称、级别和简介。

### 3.3 创作作品信息撰写
**任务描述：** 为每个创作作品撰写相关信息

**具体内容：**
1. 为10个创作作品分别撰写作品标题
2. 为每个作品撰写一段50字以内的简短描述（可选）

**操作指南：**
1. 作品标题要简洁明了，能够体现作品内容
2. 描述要突出作品的特色和创作灵感

**为什么要做这项工作：**
完整的作品信息有助于用户展示和分享自己的创作成果。

**对应界面效果：**
在创作作品卡片上显示作品标题和描述。

### 3.4 预约记录信息撰写
**任务描述：** 为预约记录撰写相关信息

**具体内容：**
1. 确定传承人姓名和擅长技能
2. 确定预约时间和时间段
3. 确定预约状态（如"已预约"、"已完成"、"已取消"等）

**操作指南：**
1. 传承人信息要与头像匹配
2. 预约时间要合理分布
3. 状态种类要覆盖常见情况

**为什么要做这项工作：**
详细的预约信息有助于用户管理自己的非遗体验计划。

**对应界面效果：**
在预约记录卡片上显示传承人信息、预约时间和状态。

### 3.5 功能菜单文案撰写
**任务描述：** 撰写功能菜单相关文案

**具体内容：**
1. 确定功能菜单项：个人资料、设置、帮助与反馈、关于我们、退出登录
2. 为每个菜单项撰写简洁明了的描述（可选）

**操作指南：**
1. 菜单项要覆盖用户常用功能
2. 文案要简洁易懂，避免歧义

**为什么要做这项工作：**
清晰的功能菜单有助于用户快速找到所需功能。

**对应界面效果：**
在功能菜单区域显示菜单项和相关图标。

## 4. 数据整理与格式化任务

### 4.1 用户数据整理
**任务描述：** 整理用户相关信息

**具体内容：**
1. 建立用户信息表格，包含字段：用户ID、昵称、头像、身份标签
2. 确保每个字段信息完整准确

**操作指南：**
1. 使用Excel或Google Sheets建立信息表格
2. 每个用户一行，字段列为不同属性
3. 定期备份数据，避免丢失

**为什么要做这项工作：**
结构化的用户数据便于技术人员开发使用，也能确保信息的一致性。

## 5. 图片与数据关联任务

### 5.1 建立图片与数据对照表
**任务描述：** 建立所有图片与对应数据的关联关系

**具体内容：**
1. 建立用户头像与用户数据的对照表
2. 建立收藏项目图片与收藏数据的对照表
3. 建立创作作品图片与创作数据的对照表
4. 建立传承人头像与预约数据的对照表

**操作指南：**
1. 使用Excel或Google Sheets建立对照表格
2. 每张图片一行，包含图片名称、对应数据ID、用途说明等字段
3. 核对关联关系的准确性
4. 整理图片版权相关信息

**为什么要做这项工作：**
确保每张图片都能正确显示在对应的数据项中，避免错位或缺失。

**对应界面效果：**
用户在浏览个人中心时能看到正确的配图。

## 6. 交付要求

### 6.1 交付内容
1. 所有图片素材按分类放入对应文件夹
2. 用户信息数据表格
3. 收藏信息数据表格
4. 创作信息数据表格
5. 预约信息数据表格
6. 图片与数据对照表
7. 所有文案内容整理成文档

### 6.2 交付时间
建议在5个工作日内完成所有素材收集和文案撰写工作

### 6.3 质量要求
1. 图片清晰度达到展示要求
2. 文案内容准确无误，无明显语法错误
3. 命名规范统一
4. 版权问题已妥善处理
5. 数据完整性和一致性良好
