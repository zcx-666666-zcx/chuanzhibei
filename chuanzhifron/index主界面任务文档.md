# index主界面任务文档

## 1. 概述

本文档旨在为非技术团队成员分配index主界面开发相关的具体任务，确保团队协作高效有序。文档将详细说明各项任务的具体内容、操作方法以及任务对最终产品的影响。

## 2. 图片素材收集任务

### 2.1 轮播图(banner)图片收集
**任务描述：** 收集5张高质量的非遗相关图片用于首页轮播图展示

**具体内容：**
1. 收集非遗进校园活动相关图片2张
2. 收集传统工艺创新大赛相关图片1张
3. 收集传承人风采展示图片1张
4. 收集非遗文化宣传图片1张

**操作指南：**
1. 通过搜索引擎、非遗保护官网、文化机构网站等渠道寻找合适的高清图片
2. 确保图片分辨率不低于1080px宽度
3. 注意版权问题，优先选用免费可商用或已获得授权的图片
4. 将图片统一命名为：banner_1.jpg, banner_2.jpg ... banner_5.jpg

**为什么要做这项工作：**
轮播图是用户打开小程序后最先看到的内容，直接影响用户的第一印象和停留意愿。优质的轮播图能有效传达小程序的主题特色，吸引用户深入了解。

**对应界面效果：**
位于首页顶部，自动轮播展示，每张图片配有简洁的文字说明，点击可跳转到相关详情页面。

## 3. 文案内容撰写任务

### 3.1 轮播图文案撰写
**任务描述：** 为每张轮播图撰写标题和简短描述

**具体内容：**
1. 为5张轮播图分别撰写吸引人的标题(不超过15字)
2. 为每张轮播图撰写简短描述(不超过30字)

**操作指南：**
1. 标题要突出重点，具有吸引力
2. 描述要言简意赅，准确传达图片内容
3. 语言风格要符合非遗文化特色

**为什么要做这项工作：**
文字说明是对图片内容的有效补充，能帮助用户更好理解图片所表达的信息。

**对应界面效果：**
轮播图底部叠加半透明蒙版，上面显示标题和描述文字。

### 3.2 推荐项目文案撰写
**任务描述：** 为每个推荐非遗项目撰写名称和简介

**具体内容：**
1. 为8个推荐项目分别撰写准确的项目名称
2. 为每个项目撰写一段50字以内的简介

**操作指南：**
1. 项目名称要准确权威，可参考官方表述
2. 简介要突出该项目的核心特色和文化价值
3. 语言要通俗易懂，避免过多专业术语

**为什么要做这项工作：**
简介文案是用户了解非遗项目的第一手资料，直接影响用户是否进一步探索该项目。

**对应界面效果：**
在推荐项目图片下方显示项目名称和简介，文字颜色与背景形成良好对比。

### 3.3 热门分类文案撰写
**任务描述：** 为每个分类撰写准确的分类名称

**具体内容：**

**操作指南：**
1. 分类名称要准确反映该类别非遗的特点

### 3.4 新闻资讯文案撰写
**任务描述：** 为每条新闻撰写标题、摘要和发布时间

**具体内容：**

2. 为每条新闻撰写摘要(不超过50字)
3. 为每条新闻设定合理的发布日期

**操作指南：**
1. 标题要有新闻性，能概括核心内容
2. 摘要要精炼准确，突出新闻要点
3. 发布时间按照倒序排列，最新的排在前面

**为什么要做这项工作：**
优质的新闻标题和摘要能显著提升用户点击率，是传播非遗资讯的关键要素。

**对应界面效果：**
在新闻列表中显示标题、摘要和发布时间，形成完整的新闻展示。

## 4. 图片处理与命名规范

### 4.1 图片格式统一
**任务描述：** 将所有收集到的图片统一为合适格式

**操作指南：**
1. 轮播图和项目图片统一为JPG格式
2. 分类图标统一为PNG格式
3. 所有图片压缩至合理大小，避免影响加载速度

### 4.2 命名规范执行
**任务描述：** 严格按照既定命名规范重命名所有图片

**操作指南：**
1. 检查每张图片的命名是否符合规范
2. 统一替换为标准命名格式
3. 建立图片清单，记录每张图片的用途

## 5. 交付要求

### 5.1 交付内容
1. 所有图片素材按分类放入对应文件夹
2. 图片清单表格（包含图片名称、用途、来源等信息）
3. 所有文案内容整理成文档

### 5.2 交付时间
建议在3个工作日内完成所有素材收集和文案撰写工作

### 5.3 质量要求
1. 图片清晰度达到展示要求
2. 文案内容准确无误
3. 命名规范统一
4. 版权问题已妥善处理
